Use Template Strings: Use backticks for better readability.


```javascript
/**
 * Demonstrates the use of template strings in JavaScript.
 * 
 * Template strings provide a more readable and concise way to combine 
 * strings and variables. In this example, we greet a user by their name
 * using template strings instead of traditional string concatenation.
 */

// Define a sample name variable
const name = "John";

// Use a template string to create a greeting message and log it to the console
// The variable 'name' is embedded directly within the string using ${name}
console.log(`Hello ${name}!`);  // Expected output: "Hello John!"

```
